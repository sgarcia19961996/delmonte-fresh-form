# Del Monte Fresh Form

## Setup Instructions

1. **Clone the Repository**:  
   Open your terminal and run:
   ```sh
   git clone https://github.com/sgarcia19961996/delmonte-fresh-form.git
   cd delmonte-fresh-form
   ```

2. **Install Dependencies**:  
   Make sure you have Node.js installed. Run:
   ```sh
   npm install
   ```

3. **Configure Power Automate**:  
   - Go to Power Automate and create a new flow.
   - Use the HTTP action to connect with the Del Monte Fresh Form backend.
   - Configure the action with appropriate URL and authentication.

4. **Run the Application**:
   To start the application, run:
   ```sh
   npm start
   ```

## Usage Instructions

- **Submit Data**: Fill out the form on the web interface and submit to trigger your Power Automate flow.
- **View Submissions**: You can view submitted data in your designated storage location.

## Troubleshooting

- If you encounter errors, check the console for messages and ensure your Power Automate configurations are correct. 
- Ensure that you have the necessary permissions to access data.
